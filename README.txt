Automated tagging and workflow management for scholarly articles.

== Description ==

This plugin automatically assigns "Progress Tags" to `scholarly_article` posts based on their data (metadata, status, author info) whenever they are saved or published.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/sad-workflow-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure rules in the settings page.

== Frequently Asked Questions ==

= How do I add a new rule? =

Go to the settings page and use the rule builder.
